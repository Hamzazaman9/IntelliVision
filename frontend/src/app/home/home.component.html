IntelliVision is a cloud-based web application which provides a smart management solution for users who deal with a big video library. Users can manage videos smartly by grouping them in-to folders and subfolders. IntelliVision will use smart video processing techniques for searching through the video for specific actions and objects without the user having to watch or skim through the video. Users of the system will be provided a cloud storage where they will upload their video library for it to be processed. User can also pass a link of a video for it to be processed. This will save the users time and effort of manually opening and searching through videos for specific content.

Subcription Plans

Basic

Basic subscription plan is for limited personal usage. It offers the following features

  • 10 GB cloud storage
  • Unlimited video processing capability
  • Smart video searching
  • Private video library

Standard

Standard subscription plan is for small businesses or coorporations. It offers the following features

  • 50 GB cloud storage
  • Unlimited video processing capability
  • Smart video searching
  • Private video library

Premium

Premium subscription plan is for big businesses or coorporations. It offers the following features

  • 100 GB cloud storage
  • Unlimited video processing capability
